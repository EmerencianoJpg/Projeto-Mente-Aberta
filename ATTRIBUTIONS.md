# Atribuições do Projeto

Este projeto utiliza os seguintes recursos de terceiros:

- Interface: Componentes baseados em [shadcn/ui](https://ui.shadcn.com/) (Licença MIT).
- Imagens: Fotografias provenientes do [Unsplash](https://unsplash.com/).
- Ícones: Biblioteca [Lucide React](https://lucide.dev/).
